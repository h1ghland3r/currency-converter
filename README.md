<br/><br/>
HTML5<br/>
CSS<br/>
VanillaJS<br/>
Fixer.io API<br/>

**Instructions for testing**
<br/>
1) Clone the project;
2) Sign up on Fixer.io and get your Free API Key for testing;
3) Open scripts.js and add your key, save the file;
4) Open index.html.
